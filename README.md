## 📋 Descrição

O **Refund** é uma aplicação web desenvolvida para simplificar o processo de solicitação de reembolso de despesas. O foco principal deste projeto foi colocar em prática conceitos intermediários e avançados de JavaScript, criando uma interface dinâmica que gerencia entradas de dados e exibe resultados em tempo real.

<br>

## ✨ Funcionalidades

- **Formulário Dinâmico**: Interface intuitiva para inserção de dados da despesa (nome, categoria, valor).
- **Validação de Dados**: Garante que os campos obrigatórios sejam preenchidos corretamente antes do envio.
- **Formatação de Valores**: Aplicação de máscaras de entrada para valores monetários (R$), facilitando a digitação.
- **Listagem de Despesas**: Exibição organizada das solicitações de reembolso cadastradas.
- **Cálculo de Totais**: Atualização automática do valor total a ser reembolsado com base na lista de despesas.
- **Remoção de Itens**: Permite excluir solicitações da lista de forma simples.
- **Uso de IDs Únicos**: Geração de identificadores exclusivos para cada item da lista (manipulação de arrays e objetos).

<br>

## :brain: Conceitos de JavaScript Aplicados

Neste projeto, foram explorados e aprofundados os seguintes conhecimentos técnicos:

- **Objetos**: Estruturação de dados para representar cada despesa.
- **Métodos de Texto (String)**: Manipulação e formatação de textos (como IDs e nomes).
- **Arrays**: Armazenamento e gerenciamento da lista de despesas.
- **Repetições e Iterações**: Uso de métodos como `map`, `filter` ou loops `for` para renderizar a lista.
- **Data e Hora**: Manipulação do objeto `Date` para registrar o momento da solicitação (se aplicável).
- **Classes** (ou funções construtoras): Estruturação da lógica de negócio e organização do código.
- **DOM**: Manipulação intensa do Document Object Model para criar, ler, atualizar e remover elementos HTML dinamicamente.
